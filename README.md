# Genome-scale metabolic reconstruction of the symbiosis between a leguminous plant and a nitrogen-fixing bacterium

## Project Description
This repository contains the models, Flux Balance Analysis (FBA) scripts, model files, and simulation reports for the analysis of the nitrogen-fixing bacterium. The project focuses on a constraint-based modelling workflow to compare the metabolic behaviour of the same bacterial models across three distinct ecological controls: Bulk Soil, the Rhizosphere, and the Root Nodule. By comparing these environments, we aim to identify the metabolic shifts required for the transition from a free-living state to an efficient nitrogen-fixing symbiont.

## Objectives
- Compare the metabolic flux distributions of the bacterium in three distinct environments.
- Identify essential reactions and carbon source preferences unique to each environmental stage.
